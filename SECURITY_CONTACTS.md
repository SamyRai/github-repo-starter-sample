# Security contacts

This document lists the recommended security contacts and basic rotation notes.

- **Primary security contact**: [security@glpx.pro](mailto:security@glpx.pro) (canonical for template)
- **Backup contact**: [security-backup@glpx.pro](mailto:security-backup@glpx.pro)
- **Rotation**: maintainers should review contact list every 6 months and verify that keys and contact emails remain valid.

Notes

- If you maintain a PGP key for the project, add the `PGP_KEY_URL` in `TEMPLATE-VARS.md` and ensure the public key is published at that URL.
- Update `SECURITY.md` to reference encryption and the project's key ID.
