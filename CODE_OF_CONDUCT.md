# Code of Conduct

This project follows the Contributor Covenant Code of Conduct.

## Our Pledge

We expect every community member to treat others with respect and
professionalism. Harassment, discrimination, and abusive behavior are not
tolerated.

## Reporting & Enforcement

If you experience or witness violations, report them to
[conduct@glpx.pro](mailto:conduct@glpx.pro). Reports will be reviewed by at
least two maintainers and handled confidentially. Retaliation against
reporters is not permitted.

For the full policy, see the [Contributor Covenant](https://www.contributor-covenant.org/).
